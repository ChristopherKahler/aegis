| Decision | Rationale |
|----------|-----------|
| curl\|bash as public install method | Zero dependencies, industry standard (syft/grype/trivy all use it) |
| Python venv over pipx/pip | PEP 668 blocks pip on modern Ubuntu/WSL; venv always works |

| Docker scanner image detection | User's SonarQube runs via MCP server using `sonarsource/sonar-scanner-cli` Docker image |
| Auto-skip installed tools | Re-running installer skips tools already on PATH |
